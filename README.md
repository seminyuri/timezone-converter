# Конвертер часовых поясов для психолога

Простой и красивый веб-инструмент для конвертации времени между часовыми поясами. Создан специально для психологов и других специалистов, работающих с клиентами из разных регионов России.

## Возможности

- **Простой интерфейс**: Выберите дату, время и город клиента — мгновенно получите результат
- **Поддержка всех регионов России**: От Калининграда до Камчатки
- **Визуальные индикаторы**: Иконки времени суток (утро, день, вечер, ночь)
- **Копирование в буфер**: Одним кликом скопируйте время для отправки клиенту
- **Недавние города**: Быстрый доступ к последним 5 выбранным городам
- **Темная тема**: Комфортная работа в любое время суток
- **Сохранение настроек**: Ваши предпочтения сохраняются между сессиями
- **Адаптивный дизайн**: Работает на телефонах, планшетах и компьютерах
- **Без регистрации**: Никаких аккаунтов, всё работает локально

## Как использовать

1. Откройте `index.html` в браузере (или перейдите по ссылке GitHub Pages)
2. Выберите дату и время вашей встречи
3. Выберите город клиента из списка
4. Результат отобразится автоматически
5. Нажмите кнопку "Копировать время" для быстрой отправки клиенту

## Развертывание на GitHub Pages

### Вариант 1: Через веб-интерфейс GitHub

1. Создайте новый репозиторий на GitHub (например, `timezone-converter`)
2. Загрузите файл `index.html` в корень репозитория
3. Перейдите в Settings → Pages
4. В разделе "Source" выберите ветку `main` (или `master`)
5. Нажмите "Save"
6. Через несколько минут сайт будет доступен по адресу: `https://ваш-username.github.io/timezone-converter`

### Вариант 2: Через командную строку

```bash
# Инициализация git репозитория
git init
git add index.html README.md
git commit -m "Initial commit: timezone converter"

# Создайте новый репозиторий на GitHub и замените URL ниже
git remote add origin https://github.com/ваш-username/timezone-converter.git
git branch -M main
git push -u origin main

# Включите GitHub Pages в настройках репозитория
```

### Вариант 3: Использование других платформ

#### Netlify
1. Перетащите папку с файлом на [netlify.com/drop](https://app.netlify.com/drop)
2. Готово! Получите ссылку вида `https://случайное-имя.netlify.app`

#### Vercel
```bash
npm install -g vercel
vercel --prod
```

## Технические детали

- **Библиотека**: [Luxon](https://moment.github.io/luxon/) для работы с датами и часовыми поясами
- **Размер**: ~15KB (HTML + CSS + JS в одном файле)
- **Зависимости**: Только Luxon через CDN, никаких установок npm не требуется
- **Браузеры**: Работает во всех современных браузерах (Chrome, Firefox, Safari, Edge)
- **Хранение данных**: LocalStorage для сохранения темы и недавних городов
